[Sơ đồ 3: Sơ đồ luồng dữ liệu và sự kiện giữa các Manager]
-----------------------------------------------------------
sequenceDiagram
    participant UI as GameUI
    participant Game as GameManager
    participant Level as LevelManager
    participant Spawner as WaveSpawner
    participant Enemy as Enemy Unit
    participant Castle as Castle Instance
    
    Level->>Spawner: Send WaveData
    Spawner->>Enemy: Instantiate & Set Target -> Castle
    Enemy->>Enemy: Moving based on A* Pathfinding
    Enemy->>Castle: AttackHit! (TakeDamage)
    Castle->>Game: HP <= 0 (LoseGame)
    Enemy->>Enemy: OnDeath Event
    Enemy-->>Spawner: aliveEnemies--
    Spawner->>Level: Current Wave Clear
    Level->>Game: All Waves Clear (WinGame)
    Game->>UI: Show Win/Lose Panel
